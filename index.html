Vite/NPM project setup w/ customTemplate
  1. npx degit z4hi/my-vite-template#main [projectname]

    - clone vite template from github

  2. Update vite git config & readme.md | delete dist folder in gitignore

  3. npm install
  4. npm run dev
  5. npm run build

  6. Create GH repos and and push to GH

  7. git add dist && git commit -m "Initial dist subtree commit"

    - add dist folder as subtree

  8. git subtree push --prefix dist origin gh-pages

    - push to subtree branch
